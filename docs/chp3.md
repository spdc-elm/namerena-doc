# Chapter 3 交流平台与提问的艺术

名竞玩家的主要交流平台有：

- [百度贴吧](https://tieba.baidu.com/f?kw=%E5%90%8D%E5%AD%97%E7%AB%9E%E6%8A%80%E5%9C%BA)（较不活跃）
- QQ 群（群号：1079992837，较活跃）
- [名字竞技场论坛](https://namerena.flarum.cloud)（较活跃）

如果有疑问可以在这些交流平台进行提问。

但注意，希望你能确保你提出的问题的答案并没有在该教程中提到，或者认为该教程的表述有歧义/难以理解，这会促使我们对教程内容进行修改。否则，我们可能不予回答，甚至处以删帖/禁言的处罚。

在提问题时，请把问题描述清楚。如果是操作类的问题，最好能附上截图等。

也最好不要问“是不是怎么样就可以了”的问题，很多时候自己试一试的成本不比提问大。

**基础的教程到此结束了，但是建议你继续阅读进阶教程了解更多。**

**附录：常用术语/俗语补充表**

为了让你能无缝融入社区讨论，这里整理了一些教程中未提及但在社区讨论中较为常见的术语/俗语：

| 术语/俗语 | 解释 |
| :--- | :--- |
| **R 大** | 游戏作者 Rinick |
| **七围** | 一个号除 HP 外的七项属性之和 |
| **虚高/花瓶** | 实战明显弱于其官方评分水平的号 |
| **杂王** | 贬义，指那些掺杂无用技能使强度降低的号（如杂雷击的背刺号） |
| **帮测** | 自己给其他玩家测号的行为 |
| **互测** | 两名玩家互相给对方帮测的行为 |
| **rp** | “人品”的缩写，指测号运气，常用动词搭配“吸 rp” |
| **赛运** | 号在比赛中的发挥（发挥超常=赛运好） |
| **签运** | 号在比赛对阵抽签中结果的好坏 |
| **猎豹** | 意为“断档级别的逆天强度”，也可用于形容离谱的 rp/赛运/签运等 |
| **劣爆** | “猎豹”的反义谐音，用于形容差得离谱的 强度/rp/赛运/签运等 |
| **桂** | 号在发动垂死抗争后获得的属性增强，也可以指号在比赛中的神操作 |
| **入/入机/软** | 号在比赛过程中的迷惑操作，如明显索敌错误、伤害过低等 |
| **抽奖** | 号在狂暴 debuff 下随机索敌的过程 |
| **使魔父亲/幻影父亲** | 召唤物的猎豹操作，如大量输出/承伤/救主等，也可用于反衬本体的入机 |
| **符木** | 高护身符熟练度的号在比赛中却没有发动护符，一击即倒的情况 |
| **不等式/镜子** | 比赛中胜率高的一方反而输掉的情况 |
| **陷阱/五五开陷阱** | 比赛中双方胜率接近五五开但实际比分一边倒的情况 |
| **拜谢** | 一个 QQ 表情包的名称，引申为“膜拜” |
| **虚空拜谢** | 贬义，在无关场合无端拜谢的行为 |
