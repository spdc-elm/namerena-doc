首先背刺担当<br>
背刺技能，配合仅有53的攻，伤害输出一点也不小。就算不能每次都秒杀对手，也基本也只打剩最后一击的血量。<br>
实力评估时潜行112%背刺55%，只有一半成功，但是在天梯实战中，配合了队友吸引火力，再加上对手通常高智力会挑挨打担当来打，成功率至少3/4。威力非常大。<br>
60%的魅惑，从结果上来看，魅惑的存在意义没有背刺大，因为魔攻不到90，遇到抗高敏高对手，被回避的概率还是很大的。但不管怎么样也是有正面意义的实力加分。<br>
39%的分身是想当厉害的。分身的特点是减属性不减技能，其他角色是 7成属性+3成技能的话，背刺担当是3成属性+7成技能。所以属性减少对他的实力影响不大，照样发动背刺和魅惑。<br>
分身以后背刺魅惑效果就算没有加倍，也有1.5倍。但是缺点是血少，挨打担当血多时无法再为他们吸引火力<br>
21%的加速，对背刺当然有很大好，加上挨打担当的97%的加速，加速和背刺一样几乎是每局必出的技能。<br>
10%垂死抗争，这个和分身结合效果应该很不错，但是未必能发动就是了。

挨打担当。<br>
24%的护身符非常猛，差不多意味着他死的时候有快一半的概率会复活。<br>
38%防御，靠这两个技能，还有339的血量，使这个防低角色也能做肉盾，看战斗记录的话会发现挨打担当死的并不是很快。<br>
98%的加速，和背刺配合加成太大。可能是他被选为配合的最主要原因。<br>
40%的连击，嗯挨打担当自己96的攻，超过80的速和敏，攻击输出也是顶级角色的水平。<br>
21%雷击，魔太低，这个可能是个鸡肋技能。

这个组合最大的重点还是挨打担当吸引火力提高背刺成功率。<br>
