# Chapter 2 实战特点与环境分析（上）

## 2.1 总览

在入门教程 2.4 节中，我们初步提及了直接有关于『实战环境』的『超强单』与『超强评』（以下取首字母简称为 cqd 和 cqp）及其测试方法，原文如下：

- 通过强评和强单，我们可以大致知道一个号的组队能力和单挑能力。但值得注意的是，它们反映的是对天卫的组队水平和单挑水平，和实际与大部分玩家对战的情况并不一样。因此我们可以通过收集主流玩家的单挑号或者多人组作为“靶子”，测对这些号/多人组的胜率的平均值，来获得更合理的实战水平评估。

很显然，为了更好反映当下的总体环境，靶子是在并且需要不断更新着的。这一方面是由于玩家总体测号量的增加，另一方面也是源于玩家游戏理解和技术水平的提升（主要是双人侧）。接下来两篇内容的目的即让读者从靶子的产生与发展史出发，逐步了解一些较为主流的克制关系，获得新的理解。

注：所谓克制定义为，两『类』『环境胜率接近』的号/组，一方对对方的胜率有明显对五五开的偏离。以下不考虑强度差异过大情况下的胜率关系，也选择性忽略极端非主流技能组相关的胜率分布。

在更详细的了解发展史之前，我们希望读者能对名竞发展的时间线有一个简要的了解。我们从 2020 年寒假（疫情期间）名竞社区的正式完善建立开始说起（此前名竞社区人数极少，相互之间也很少交流）。我们将用 → 连接几个关键事件，不作详细展开。一些具体时间节点已难以考证，只能表示出先后关系。

**前天梯时期**：玩家【新纪元】测出超强号，引发关注 → 玩家【曙光】提议并成功举办了一次名竞单挑比赛 → 玩家【云剑】尝试联系作者【Rinick】并成功 → 作者【Rinick】 更新了测胜率功能 → 玩家【云剑】提出了基于胜率的衡量一个号单挑实力强弱的方法 → 作者【Rinick】预告可能会更新遭遇战模式（实际上更新了天梯）。

**官测和量子波动时期**：2020 年 5 月，作者【Rinick】进行了第一次削分身并正式发布第一代天梯，当时只有单人、双人和五人组 → 随着按键精灵技术的逐渐完善，作者【Rinick】更新了官方测号器 → 作者【Rinick】更新天梯为六榜（1a/1b/2/3/4/5 人组），并细化了四种评分类型 → 基于官方测号器的文哥测号器被玩家【文哥】发明 → 武术大赛开始举办 → 不依赖官方测号器的速度更快的量子波动测号器被很多玩家共同发明并完善。

**核时代前期**：核测号技术被发明并公开（2021 年 3 月）→ 分身强号被大量挖掘，随后作者【Rinick】进行了第二次削分身（2021 年 8 月）→ 很多玩家因为各种原因逐渐淡坑，名竞进入大淡季 → 天梯因为服务器欠费关闭，至今未再次打开 → 2023 届高考结束，【新纪元】【曙光】【Squall】等老玩家回坑，名竞重新进入旺季。

后面的事后面再说。

## 2.2 单人组环境发展史

### 2.2.1 上古时代与群魔乱舞

在远古的名竞中甚至没有『单挑评分』乃至『胜率测试』一说，直到2020年，大量玩家涌入名竞圈，天梯开放，才出现了以上两者。当时普遍衡量单挑强度的方法就是强单（qd）数值。少数较为重视天梯排名的玩家会花费力气测试自己不同单挑号对天梯前排强号的胜率均值以决策出场阵容——这被认为是超评分系列想法的起源——但由于当时这种方法十分缓慢（尚未开发出加速评分网站）、无法自动化（没有集成工具）、天梯前排强号随时可变、其余玩家并未重视等种种原因，『靶子』系统未能建成。

这里补充一点，胜率测试这个功能的出现略早于天梯。在此段间隙内玩家【云剑】构筑了一个最初由所有公开的普评（当时评分并未细化，只有这一种评分）6800+的号组成的靶子，用对战他们胜率的平均值作为衡量单挑实力的依据。由于靶子的选取方式比较随意，并且测试需要的时间较长，在天梯建立后逐渐被人遗忘。当时的人们更倾向于用天梯排名评价实战强度（即，将待测号放上天梯，待几天后排名趋于稳定，并以此衡量）。

### 2.2.2 核时代前中期与净化的统治

随着核测号器的发明，玩家们的单挑号强度得到了明显的提升。然而，核时代早期的单挑强度评价体系仍然被qd与天梯排名两大因素主宰，与量子波动时期并无本质区别。当时也有玩家在实践中意识到qd数值与天梯排名并不严格相关，初步形成了『虚高/花瓶』『虚低』的概念，但对于引起这一现象的原因和容易虚高/虚低的技能种类等问题，则并没有进一步的研究。

相较于量子波动时代高qd单挑号的群魔乱舞，核时代的高端单挑几乎清一色是净化号（你到现在的qd十大榜单去看，也都是净化），其最核心原因在于，qd测试和天梯排名头部面对的是技能五花八门的天卫，净化可以锁住它们的技能，从而有了相当的优势。环境中第二多的是幻术号，当然也不乏少量相对冷门的技能，它们共同构成了净化占半边天的0代单人组环境。

核时代中期，在经历了漫长的大淡季后又有不少新人入坑与老玩家回归。届时天梯已经倒闭，但群友不断组织的各类比赛撑起了社区的活跃度，超评分思维也是在这样的社区环境中再次萌芽。当时也有胜率测试，不过大多都只是对最顶尖的几个号测试胜率，打过就是赢。这种做法过分关注打零星几个号的优劣，却忽略了『大环境』的存在。

### 2.2.3 两极的形成与靶子的初创

随着测胜率的人越来越多，大家逐渐开始意识到，qd与真实的单挑实力之间存在不小的差距。举个例子，一些qd6700的幻术就有接近于7100净化的单挑水平（对打与打别的号都是如此）。在这样的事实基础下，社区开始有了净化虚高约400的说法，比赛大环境中幻术的数量也逐步提高到与净化接近相当的水平。这使得净化与幻术成为了单挑界的『两大技能』，而『虚高400说』也让群友在心中默认了同实力净化与幻术之间不存在明显克制，『净幻两极』格局初步建立。

与此同时，当时的极少数胜率分布相当奇特的号，比如『vRuH:z@耗子尾汁』『Iwn<Zo@nan』，却被单拉出来列为『净化杀手』，颇有一种异端的感觉。与之相对的自然还有『幻术杀手』这一说法（这一说法的提出比净化杀手要晚，原因是早期并无典型例子，真要说的话可以举例内战型幻术『SINGULARITY \#rfExlcF@powerless』）。

手动测试胜率的方法在强号少的时候还管用，大家都大致找到了自己所谓『实战强』的几个号，但是这显然不是长久之计。终于，在各种因素的推动下，于2023年9月30日，玩家【曙光】提出了制作一个号库用来存放大家的『实战强号』，并采用脚本测试平均胜率作为评判实战依据的方法。自此，初代cqd靶子与1代单人组环境正式形成。

### 2.2.4 环境的崩坏与构建克制理论的失败

靶子形成以后的几个月里，几乎所有玩家都将这样的『cqd』数值奉为圭臬，似乎忽略了靶子本身的片面性与停滞性。而另一方面，各种并非热门技能组的单挑号如雨后春笋般冒出，有瘟疫『wt\(zEe\[T@TigerStar』，背刺『Garakuta tKjEzvOfSnnJCb@Squall』，分身护符『CtC7gL9JNpvNpOl@新纪元』等，无一不是对现有靶子环境的挑战。

这时候的玩家在干什么呢？他们并没有意识到问题的严重性，正在尝试构建『两极四象』的克制理论，即『净化→幻术杀手→幻术→净化杀手→净化』，形成的克制闭环：

- 玩家【新纪元】的定性解释是：① 净化与幻术互相没有明显克制；② 净化杀手克制净化，却被幻术克制，主要包含高八围物攻、诅咒、护盾、魅惑等技能；③ 幻术杀手克制幻术，却被净化克制，主要包含隐匿、吸血、聚气、背刺等技能；④ 从基本的净化/幻术出发，增加哪种杀手的技能就更偏向于哪个杀手方向（相比cqd对该类胜率增加，对另一类降低）。上述观点刻意避免考虑冷门技能，虽说讲得没有太大问题，但却是相当欠完善的。
- 玩家【Squall】则试图用极坐标来定量分析，对同强度的不同类型号赋予不同的辐角值，通过计算角度差的正弦值来判断克制关系。这种观点下四象图正好排成一个圈，但是通过数据拟合发现对于那些冷门技能完全无法给出正确的判断。比如背刺克制除了净化外的三者，对模型而言就是数据污染。

通用理论的建立尝试宣告破产，环境也早已慢慢偏离了六个月前的靶子。唯一的办法就是更新换代，于是出现了加入那些『冷门技能』号的第2代环境。

### 2.2.5 二代环境的双重失衡与新靶子的机器构筑

统计学意义上讲，二代靶子已经涵盖了几乎所有不同胜率分布类型的号，理应不那么敏感。但事与愿违，也许是整个名竞的出号背离了统计规律，在靶子换代后的几个月里，背刺与低抗净化这两种幻术杀手层出不穷，给了当时版本的幻术非常大的压力（同时对普通净化而言是利好）。随时间推移各战队整体实力也有了提升，不少当初的小队和新人都有了不容忽视的门面强号。这种环境对类型和玩家的双重偏移也是催生了第三代（即如今）的靶子。

有了前两次做靶子的经验与几次武术大赛环境与靶子严重不符的教训，新靶子的制作从『玩家的决策』出发，尽可能地模拟真实环境。因此我们把任务交给了机器，通过模拟每个人选择自己在当前环境cqd最高的几个号并多次迭代来构筑相对能稳定更久也更切合实际的环境。从现在的眼光看来，质量确实高于前面两个版本，但也由于某些高方差号的出现，环境有了摇摇欲坠的迹象。当然靶子应该是不断进化的，至于第四代究竟会是什么样，我们交给时间。

## 2.3 单挑号的基本克制关系

写在前面：名竞一个号有属性+技能共四十余个参数，要想解释克制关系最好的方法肯定是模型拟合（当然也远未成功），简明地讲述必然漏掉一些特例情况。所以下面仅仅从战斗角度分析各大主流技能的特点。

### 2.3.1 净化与净杀（净化杀手）

**净化的技能特点**：<br>
100%命中率，清除对方buff，减少对方之后使用技能的频率，对召唤物有额外增伤。

**净化号的战斗特点**：<br>
净化控场的局（指先手净化，不完全等同于优势局），对方受控制一般不会有过多主动技能释放，通常为较为简短的对A局，也因此每回合都尤为重要，八围数值成为了相当重要的因素。

**净化号克制的技能**：<br>
克制buff类技能（聚气，蓄力，加速，无盾铁壁），克制八围不高的大多号（尤其是抗和攻低的，对A中就这两项最重要，抗低同时包括幻影抗低的）。<br>
事实上净化的克制面相当广，包括很多法师和杂号（天卫），也不那么容易被针对，所以无论环境怎么变化，净化总有一席之地。

**克制净化的技能**：<br>
通过上面的分析，目前战胜净化无非两条路，一是争取先发制人创造足够优势（瘟疫，诅咒），二是在朴实无华的对A中占据上风，或是增攻减伤（高八围，会心，垂死，高护盾和护盾铁壁），或是抢回合（护符，反击）。目前的净杀基本都是这些技能或是它们的组合。

**反制净杀的手段**：<br>
强净化或多或少都需要一些对克星的抗性。对净化而言，不存在什么血脉压制，这并不那么困难。对于瘟疫/诅咒，可以选择命轮后发制人；对于护盾，可以搭配少量聚气/蓄力破盾；当然也可以选择控场后使用吸血/加速/减速增加对A胜率；最后，还有最通用的方法——内卷八围，强化数值总是有利无害的。

通过上面的讨论可以发现，在净化号的技能组中掺杂一些有用的副技能，对其单挑强度是有利的。这种『杂净化』占优的环境促使一批『连控型净杀』出现，即在对A中趁着净化号释放杂技能的间隙快速回复自身的技能发动频率，并使用高频冰冻术等控制技能反转攻守，将净化号一路控死。这种净化越杂越能杀的新型净杀也对环境中的净化单挑起到反向选择作用，形成了一个有趣的协同进化过程。

### 2.3.2 幻术与幻杀（幻术杀手）

**幻术的技能特点**：<br>
召唤3回合后行动的幻影，幻影面板相当于血量减半的独立号（与本体属性不同），除了平A以外会附体（熟练度比较高），附体成功后自身消失，对方狂暴4回合。

**幻术号的战斗特点**：<br>
幻术控场的局（指先手幻术展开）幻影数量一般都多于本体（本体回合也更有可能选择继续展开，虽然熟练度下降），靠着人海优势淹没对方。因而幻影本身的强度也将会非常重要。对一个强幻术单挑的要求是相当苛刻的，本体面板和幻影面板都至关重要。以下内容主要针对本体展开和偶尔的抗伤，幻影输出的主流幻术/分身幻术。<br>
首先对于本体，展开需要幻术真实熟练度（因此与净化不太一样，杂技能越少的幻术相对展开越快，这是一个少即是多的优势）与速和智，抗伤需要防抗HP，面对最主流的单挑净化的时候要做好对A准备，需要攻和敏。本体只有魔没用，这也成为了一个舍弃项。<br>
而对于物攻为主的幻影，HP和攻防速敏显而易见的重要。面对法师抗也不能低，否则很容易被一发一个。魔和智相对也没用，尤其后者更多是反作用：智高的幻影更容易发动附体，不仅造成了得不偿失的输出位减员，在隐匿防狂暴逐渐普及的环境中，狂暴debuff也往往收效甚微（即使对方不防狂暴，在人数众多的场上随机索敌打中自己的概率也很小）；附体的低命中率更是使得对方很容易回避掉，白白浪费了行动机会。因此，多数幻术号的幻影智都较低。以及最后一点，要求幻影的嘲讽值高于本体，否则幻影无法起到承伤的作用（这对绝大多数的幻术都成立，特例暂不讨论）。<br>
综上，幻术号对八围属性的要求比别的号多了好几种，也因此幻术的理论上限非常高，目前测号量级出的幻术单挑都或多或少有着较为明显的缺陷，可能会出现在各个方面。

**幻术号克制的技能**：<br>
首先对于内战，人数优势就是胜利，你得争取在你的回合召唤尽可能多的幻影，所以上文提到的纯技能优势就体现出来，幻影质量也很重要，分身幻术展开更快也有一些优势。<br>
而对于别的技能，克制明显的就是主修对A的类型：最明显的是魅惑，因为幻术的技能效果不受魅惑debuff影响；其次是对A防御流（护盾铁壁），由于其输出伤害不足的缺点，一旦你完成展开，他就只能徒劳地打幻影，毫无招架之力；有克制但不那么明显的是是对A进攻流（诅咒，瘟疫，减速），如果对方无法抢在你展开之前一套连招秒杀你，那你展开了还是你的主场（由此可见嘲讽值的重要性，如果本体嘲讽更高，即使展开了也会被精准命中斩杀）。此外值得一提的是逐渐增加于环境中的分身护符，它的展开一般没有幻术更快，而且由于分身对属性的削弱，这也成为了幻术克制的对象。

**克制幻术的技能**：<br>
内战克制已在前文中分析。面对幻术号，如果你是净化先手控制并且对方存在低攻等面板劣势就相当致命（尤其需要指出，幻术体系没有魔攻，所以净化号的抗低并不影响打幻术胜率，因此在内战弱的映衬下，低抗净化也被认为比同水平的高抗净化更加克制幻术），这是其一；如果你无法阻止对手展开可以使用聚气增伤，不仅可以更快速清除幻影，对本体输出速度也是优势，这是其二；最后，幻影HP平均是本体一半的特性和对方展开幻影并滞后行动的输出窗口期，都给了背刺的特殊机制（索敌更重视HP而非嘲讽值）一发入魂的机会，这是其三。另外，防狂暴的隐匿当然也是一点加成。

读到这里不难发现，幻术克制的技能多为前一小节所列的净杀技能，而幻杀技能也主要是上面提到被净化所克制的，在目前的测号量级下似乎达成了四角微妙平衡，这也就是当初『净化-净杀-幻术-幻杀』四环与辐角理论的支撑点。

**反制幻杀的手段**：<br>
幻术的强度远没有触及理论瓶颈，通过diy定制捏假人实验可以得到强度相当妖魔鬼怪却具有合法属性的幻术单挑，面板属性当然是一条内卷的阳关道。在本体技能方面，掺杂其他主动技能有减缓展开速度的副作用，而被动技能就不存在这个问题，所以增加一些被动技对于幻术号显然相当有用，比如防御护符抗背刺，反击垂死增强对A等，这里不过多赘述。
